**Employee Payroll Management System** is a Java-based console application designed to efficiently manage employee details, attendance records, and payroll calculations.

**Features**
1. Employee management (add, view employee details)
2. Attendance tracking
3. Payroll calculation based on attendance
4. Clear separation of responsibilities using OOP concepts

**Technologies Used**
1. Java
2. Object-Oriented Programming (OOP)
  * Classes & Objects
  * Encapsulation
  * Method interaction

**How the Project Works**
1. The program starts execution from `EmployeePayrollSystem.java`
2. Employee details are managed through `EmployeeManagement`
3. Attendance records are maintained using `AttendanceManagement`
4. Salary is calculated using `PayrollCalc`
5. Final payroll output is displayed in the console
